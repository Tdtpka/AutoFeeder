Dự án thiết kế sản phẩm "Máy cho thú cưng ăn điều khiển qua ứng dụng di động", sử dụng ESP32 và framework Flutter.

Công nghệ sử dụng:
- Phần cứng:
  - ESP32
  - HX711
  - Servo SG90
- Phần mềm:
  - Framework Flutter.
  - Dart.
  - Arduino IDE.
  
Tính năng:
- Điều khiển trực tiếp qua ứng dụng di động.
- Đặt lịch qua ứng dụng.
- Xem lịch sử cho ăn.
- Theo dõi lượng thức ăn.
